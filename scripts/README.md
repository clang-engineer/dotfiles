# Scripts

`chezmoi apply` is the entry point that sets up the environment, and `scripts/` is the **tooling** around it —
some are invoked by chezmoi run-scripts, but most are helpers you run manually when needed.

## Structure

```
scripts/
├── README.md
├── .secrets.example / .secrets.ps1.example   # ~/.secrets template
├── .gitconfig.local.example                  # git identity template
├── .pgpass.example                           # ~/.pgpass format reference (real file overlaid by secrets repo)
├── generate-key.sh / .ps1                    # SSH key generation + ssh-agent registration
├── add-workspace-user.sh / .ps1              # per-workspace git identity (includeIf)
├── setup-secrets.sh                          # ~/.secrets interactive customizer (optional)
├── setup-java-versions.sh                    # jenv-based Java version setup
├── tmux-layout.sh                            # tmux layout helper
├── lib/                                      # shared functions (common.sh)
└── windows/
    ├── install-windows.ps1                   # Windows install orchestrator
    ├── install-packages.ps1                  # Scoop package install (packages/scoop-packages.txt)
    └── update-terminal.ps1                   # Windows Terminal font setup
```

## Execution summary

| Purpose | macOS/Linux | Windows |
| --- | --- | --- |
| Full environment setup | `chezmoi apply` | `chezmoi apply` |
| Windows-only install | — | `.\scripts\windows\install-windows.ps1` |
| SSH key generation | `./scripts/generate-key.sh <label> <email>` | `.\scripts\generate-key.ps1` |
| Per-workspace git identity | `./scripts/add-workspace-user.sh` | `.\scripts\add-workspace-user.ps1` |
| Java versions (jenv) | `./scripts/setup-java-versions.sh` | — |

> In `run_after_30-secrets-overlay.sh`, chezmoi reads `SECRETS_DIR` from `~/.secrets`
> and automatically runs the `secrets` repo's overlay (ssh hosts, nvim DB connection).
>
> The `.nvim.lua` for the Neovim Java LSP is generated by the nvim command `:JvmEnvInit`
> rather than by a shell script (`nvim/docs/java-lsp.md`).
